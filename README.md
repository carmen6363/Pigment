# RDA Pigments
Statistical analysis related to the manuscript "Tracking striking algal changes over the last~ 400 years using subfossil pigments in a high mountain lake (Sierra Nevada, Spain): Have we entered an unprecedented era?" https://doi.org/10.5194/egusphere-2025-5229.
R codes for stepwise RDA analysis is shown.
pigment variable refer to the matrix of the carotenoid pigments, it is de dependent matrix variable. ambien variable refer to the explicative matrix variable with MATA, APA and SPI variables. 
Pigments are Hellinger transformed and climate variables are relative scale transformed.
